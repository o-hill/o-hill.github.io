Microsoft Software Engineering Intern

During the Summer of 2018 I worked in Redmond, WA at Microsoft as a Software Engineering Intern in the Azure Networking team. Azure being Microsoft's cloud architecture, there was, and is, a lot of work to be done to organize the pieces of it. The Networking team is in charge of both inter- and intra-data center system networks, and has a lot of responsibility over the production level software in Azure.

My project there was architecting and building out a distributed packet capturing system across all of Azure to enable developers to debug on the network more efficiently and effectively. I also took part in a project to develop a solution to highly latency-sensitive data streaming on the Azure platform using Internet Group Management Protocol, which is similar to a network-wide publisher-subscriber model.

While I enjoyed my time at Microsoft, I found myself looking forward to the research seminars I was going to nearly daily, and not the work that I was doing. I felt uninspired by the corporate feel, and by my work. This pushed me to realize that I want to pursue a career in something that matters more to me, in a setting that I find compelling.
